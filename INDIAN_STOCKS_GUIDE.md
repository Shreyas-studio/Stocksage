## How to Add Indian Stocks

Indian stocks are traded on **NSE (National Stock Exchange)** and **BSE (Bombay Stock Exchange)**. To track Indian stocks in this app, you need to use the correct stock symbol format.

### Stock Symbol Format

**For NSE (National Stock Exchange):**
- Add `.NS` suffix to the stock symbol
- Example: `RELIANCE.NS`, `TCS.NS`, `INFY.NS`

**For BSE (Bombay Stock Exchange):**
- Add `.BO` suffix to the stock symbol
- Example: `RELIANCE.BO`, `TCS.BO`, `INFY.BO`

## Example: Adding Reliance Industries

1. Click **"Add Stock"** button
2. Enter symbol: `RELIANCE.NS`
3. Enter quantity: `10`
4. Enter buy price: `2500`
5. (Optional) Set target sell price: `2800`
6. Click **"Add Stock"**

The app will automatically fetch the current price from Yahoo Finance!

## Note about US Stocks

If you want to track US stocks like NVIDIA:
- **NVIDIA**: Use symbol `NVDA` (no suffix needed)
- **Apple**: Use symbol `AAPL`
- **Microsoft**: Use symbol `MSFT`
- **Tesla**: Use symbol `TSLA`
- **Google**: Use symbol `GOOGL`

But since you're using Angel One (Indian broker), you'll primarily want to use Indian stocks with the `.NS` or `.BO` suffix.

## Need Help Finding a Symbol?

1. Go to [Yahoo Finance India](https://in.finance.yahoo.com)
2. Search for your stock
3. The symbol will be shown with `.NS` or `.BO` suffix
4. Copy and use that exact symbol in this app
